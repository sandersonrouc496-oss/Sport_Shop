# SportShopProject

Консольное Java-приложение спортивного магазина.

## Возможности
- Просмотр товаров
- Добавление товара
- Удаление товара
- Поиск товара по названию
- Оформление заказа
- Просмотр заказов

## Используемые технологии
- Java
- OOP
- ArrayList
- Exception Handling

## Структура проекта
- model — модели и классы
- service — логика магазина
- ui — меню и взаимодействие с пользователем

## Как запустить
1. Открыть проект в IntelliJ IDEA
2. Запустить файл Main.java

## OOP принципы
- Инкапсуляция
- Наследование
- Полиморфизм
- Абстракция
- Интерфейсы
